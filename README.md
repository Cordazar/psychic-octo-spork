# Telenor test

Solution for the Appointments Scheduling System.

## How to use

### Add api key + url

Copy `.env.local-example` to `.env.local` and add the correct URL and KEY.

### Steps for development purposes only

`yarn husky install`

### Run it

```bash
npm run dev
# or
yarn dev
```
